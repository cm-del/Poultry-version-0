# 🐔 نظام إدارة مزارع الدواجن الشامل

نظام ويب متكامل لإدارة مزارع الدواجن يتتبع 3 عنابر مع حسابات تلقائية شاملة.

## ✨ المميزات

- 📊 متابعة 3 عنابر بشكل منفصل
- 🧮 حساب معامل التحويل (FCR) تلقائياً
- 🌡️ حساب التهوية (CFM) حسب العمر
- 📦 تتبع العلف بالشيكارة (50 كجم/شيكارة)
- 💀 متابعة النفوق ونسبته
- 💉 برنامج طبي يومي شامل
- 📈 معايير معيارية للأوزان والاستهلاك
- 💾 حفظ تلقائي في المتصفح
- 📥 تصدير واستيراد البيانات
